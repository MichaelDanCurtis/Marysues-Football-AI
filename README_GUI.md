# Fantasy Football GUI

This is a Streamlit-based GUI for the Fantasy Football Simulation Framework.

## Setup

1.  Install the required packages:
    ```bash
    pip install -r requirements.txt
    ```

## Running the App

1.  Run the Streamlit app:
    ```bash
    streamlit run app.py
    ```

2.  The app will open in your default web browser.

## Features

*   **League Overview**: View current standings and team details.
*   **Team Analysis**: Inspect rosters, projections, and strength of schedule.
*   **Matchup Simulator**: Run Monte Carlo simulations between any two teams to predict win probabilities.
